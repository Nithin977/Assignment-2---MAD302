Description

The project is designed using software programming in an Android environment. This is done using RecyclerView Layout where on clicking any item in the list, a page pops up showing details regarding that particular item.



Features

• RecyclerView Layout, Adapter, ViewHolder

• Chosen Item using lambda expression

• Moving from one activity to another using intents

• Passing data from one activity to another

• Data consistency while transferring it to DetailActivity

• A demo of six items in the list of items



Technologies Used

Language: Kotlin

Android Studio

RecyclerView

View Binding
